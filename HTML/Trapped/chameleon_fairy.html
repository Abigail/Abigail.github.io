%%PIECE%%

by Abigail

The %%PIECE%% changes the way it move on each step. Its first step, it moves like a Queen, then like a Knight, then like a Bishop, and finally like a Rook, before starting the cycle again.

Compare to the Chameleon as described by Anthony Dickins, which cycles through the same pieces, in the same order, except that one starts moving as a Knight.

Game Piece Info
Fairy Chess Chameleon

Spiral

Square

On the Square Spiral, the %%PIECE%% gets trapped after 415 steps, which is 353 steps more than the Chameleon which starts moving as a Knight.

Diamond

On the Diamond Spiral, the %%PIECE%% get trapped quicker, after 190 moves. Still 92 steps more than the Chameleon which starts moving as a Knight.

Wedge

Folded

On the Folded Wedge, the %%PIECE%% gets trapped after 73 steps, 57 steps more than the Chameleon which starts moving as a Knight.

Flat

On the Flat Wedge, the %%PIECE%% uses an 8 step loop to move to a square which lies two to the upper right from where it started the loop, giving an escape velocity of \(\frac{\sqrt{2}}{4}\). While escaping to the upper right, if visits the four right most squares of each row.

The %%PIECE%% visits the same squares as the Chameleon which starts moving as a Knight, which has the same escape velocity, but it uses a different pattern.
